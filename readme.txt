== Description ==

This plugin is used for getting all the video with title and description from your youtube channel and create that data as post into Wordpress based on that content.All you need to done is set your channel id and youtube API key from admin and click on button Sync Youtube Post it will Sync all your youtube post in to Wordpress post.

== Frequently Asked Questions ==

= How to connect YouTube =

Just enter your YouTube Chanel id and YouTube api key and click on connect button.



== Screenshots ==

1. Screenshot 1 show that how you can create your your google project in console, so that you can get your YouTube Chanel id and api key. https://console.cloud.google.com/apis/dashboard?project=youtubetowppost

2. After getting your key and id, You can connect YouTube via this screen configuration by entering Chanel id and api key and click connect button as per shown in the screenshot.

3. After connected successfully, you will see another button Sync Youtube Post when you click on that button it will fetch all YouTube post and insert into Wordpress post.
